# Hospital-Patient-Record-System
The Hospital Patient Record System is a beginner-friendly Python mini project designed to manage patient information efficiently. It allows users to add, search, update, and view patient records while storing details such as patient ID, name, age, disease, department, and prescribed medicines.
This project demonstrates the practical use of fundamental Python concepts including lists, tuples, dictionaries, loops, user input, and conditional statements. It provides a real-world healthcare scenario and serves as an excellent project for beginners to showcase their Python programming skills on GitHub.
